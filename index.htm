Project

Satellite-Based Monitoring System for Mangrove Forest Conservation

Our team developed an AI-powered satellite-based monitoring system for mangrove forest conservation in the Sundarbans using multi-year Sentinel-2 and Sentinel-5P data (2019–2024). The system calculates NDVI and CO concentrations to detect vegetation health and carbon emissions, leveraging Google Earth Engine for large-scale geospatial processing. A Random Forest model was trained to predict forest degradation and assess environmental impact. The project features an automated pipeline for daily data updates and a real-time interactive dashboard for visualizing trends, issuing alerts, and supporting conservation strategies. This solution enables data-driven decision-making for researchers and environmental authorities.
